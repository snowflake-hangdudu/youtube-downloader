需要登录 YouTube吗？

扩展不强制登录。未登录时通常只能下较低清晰度；登录后可能更高（视视频与账号而定）。

清晰度偏少时：登录 YouTube → 刷新视频页 → 再打开下载面板。

为什么最高只有 720P？

  • 可选清晰度由 YouTube与您的账号决定
  • 未登录或非Premium时，高清可能不可用
  • 部分视频本身就没有更高清晰度

下载失败怎么办？

按顺序试这三步即可：

  1. 先播放视频 5～10 秒，再点下载
  2. 改选 720P 后重试
  3. 刷新页面（F5），或在扩展管理页「重新加载」后再试

进度停在「合并音视频」很久？

高清文件较大，需要在您电脑上合成，属正常现象，请稍等。体积越大越久。下载前可看「预计大小」。

章节/列表 和右侧「播放列表列表」有什么区别？

不一样，请不要当成同一种。

  • 章节/列表：同一个视频里的多段（P1、P2、P3…）。面板会出现章节/列表 按钮，并可「队列下载全部」。
  • 播放列表 / 系列列表：播放器右侧一串不同视频（点进去会换另一个地址）。这不是章节/列表，目前请打开想下的那一集，再点「开始下载」。

普通单集视频只有「开始下载」，不会出现队列按钮——这是正常的。

章节/列表 队列怎么用？

  • 开始下载：只下当前选中的那一 P
  • 队列下载全部：按 P1 → P2 → … 逐个下载，每 P 一个文件

提示文件过大？

超大文件会提示改用较低清晰度，以保护电脑流畅。长视频建议选 720P。

支持哪些页面？

仅支持 YouTube普通视频播放页。番剧、付费专区等暂不支持。
